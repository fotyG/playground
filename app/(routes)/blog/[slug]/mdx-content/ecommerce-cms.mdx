This is one of more recent projects that helped me learn a ton about Next.js, particularly about the API routes and how it is possible to make them dynamic and secure ðŸ”¥  
The primary function of this application is to create and manage multiple stores from a single dashboard. Each store has its own unique API routes that are generated dynamically based on the store ID. For ease of use, under each section all existing routes are displayed with a button to copy the URL to clipboard. These routes are also marked as private or public based on the type of CRUD operation they provide ðŸŒŸ
